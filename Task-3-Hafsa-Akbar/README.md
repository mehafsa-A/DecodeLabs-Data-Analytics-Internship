# Project 3 — SQL Data Analysis

## Overview
Used PostgreSQL and pgAdmin to perform 
SQL analysis on 1,200 e-commerce orders.
Wrote 35 queries covering filtering, 
grouping, aggregation and sorting.

## Database
- Tool: PostgreSQL 18
- Interface: pgAdmin 4
- Dataset: 1,200 e-commerce orders

## Queries Written
- Basic exploration (SELECT, LIMIT, DISTINCT)
- Filtering (WHERE, AND, OR, BETWEEN)
- Sorting (ORDER BY DESC/ASC)
- Grouping (GROUP BY, COUNT)
- Aggregations (SUM, AVG, MIN, MAX)
- Advanced filtering (multiple conditions)

## Key Findings
- Total revenue: $1,264,762
- Cancelled orders lost revenue confirmed
- Chair leads in revenue via SQL query
- 8 outlier orders above $3,330 confirmed
- Instagram top acquisition channel
