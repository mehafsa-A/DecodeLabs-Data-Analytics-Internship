# Project 1 — Data Cleaning and Preparation

## Overview
First project of my Data Analytics internship at DecodeLabs. 
The goal was to clean a raw e-commerce dataset and prepare 
it for analysis. Clean data is the foundation of every 
reliable insight.

## Dataset
- Records: 1,200 orders
- Columns: 14 (OrderID, Date, Product, Quantity, 
  UnitPrice, OrderStatus, PaymentMethod, and more)

## Problems Found and Fixed

| Issue | Details | Action Taken |
|---|---|---|
| Missing Values | 309 blank cells in CouponCode column | Filled with "No Coupon" |
| Column Formatting | Column names had extra spaces | Standardized all column names |
| Data Completeness | Verified all 1,200 records complete | Confirmed 0% error rate |
| Duplicate Check | Checked all OrderIDs for duplicates | Zero duplicates confirmed |

## Key Result
Raw dataset had 309 missing values.
Cleaned dataset has 0 missing values across all 14 columns.

Microsoft Excel
